## 🎯 Actions Courantes

### Chercher Une Information
1. Utilisez la **barre de recherche** en haut
2. Tapez votre recherche
3. Consultez les résultats

### Télécharger Un Document
1. Ouvrez le document/photo
2. Cherchez le bouton **"Télécharger"**
3. Le fichier s'enregistre sur votre appareil

### Envoyer Un Message
1. Allez à la section **"Contact"**
2. Remplissez le formulaire
3. Cliquez **"Envoyer"**
4. Vous recevrez une réponse dans les délais

### Participer À Une Activité
1. Allez à **"Nos Activités"**
2. Consultez le calendrier
3. Cliquez sur un événement pour les détails
4. Inscrivez-vous si demandé (ou informez votre responsable)
